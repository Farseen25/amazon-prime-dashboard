# Amazon Prime Video Dashboard 📊

## Overview

This Power BI dashboard provides an interactive analysis of the Amazon Prime Video dataset. It helps users explore content distribution, genres, release trends, ratings, and geographical insights through visually appealing and dynamic reports.

## Features

* Total number of Amazon Prime titles
* Movies vs TV Shows comparison
* Content distribution by country
* Genre-wise analysis
* Release year trends
* Rating distribution analysis
* Interactive filters and slicers
* Dark-themed dashboard design

## Dashboard Insights

### Content Distribution

* Displays the total number of titles available on Amazon Prime Video.
* Compares Movies and TV Shows using visual charts.

### Genre Analysis

* Identifies the most popular genres on the platform.
* Helps understand content preferences and trends.

### Release Year Analysis

* Shows how content production has evolved over time.
* Highlights growth in content releases in recent years.

### Country-wise Analysis

* Visualizes content availability across different countries.
* Helps identify regions with the highest content contribution.

## Tools & Technologies

* Power BI Desktop

## Dataset

The dataset contains information about Amazon Prime Video titles, including:

* Title
* Genre
* Country
* Rating
* Release Year
* Duration
* Cast
* Director
